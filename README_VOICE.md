# 🎙️ Finora Voice Chat Integration

This project now supports **two modes** of voice interaction:

## 1. 🌐 Native Browser Engine (Zero Token / Free)
Built-in support using Chrome's Speech Recognition and Synthesis API. 
- **Cost**: 100% Free
- **Tokens**: No LiveKit keys required.
- **Backend**: Uses your existing Supabase Edge Functions (`chat`).
- **How to use**: Click the Microphone icon -> Settings -> Select "Native Engine".

---

## 2. ⚡ LiveKit Master Voice Chat (Advanced)
A professional, low-latency real-time voice AI system.

### 🏠 Local Setup (Complete the Backend)
To run the LiveKit Voice Agent locally:

1. **Install Python Dependencies**:
   ```bash
   cd voice-agent
   pip install -r requirements.txt
   ```

2. **Set up .env in voice-agent folder**:
   Create `voice-agent/.env` with:
   ```env
   LIVEKIT_URL=wss://your-project.livekit.cloud
   LIVEKIT_API_KEY=your-api-key
   LIVEKIT_API_SECRET=your-api-secret
   OPENAI_API_KEY=your-openai-key
   ```

3. **Run the AI Agent**:
   ```bash
   python agent.py dev
   ```

### 🔗 Connecting Frontend
Once the agent is running, update your project’s root `.env`:
```env
VITE_LIVEKIT_URL="wss://your-project.livekit.cloud"
# The token will be auto-generated by your Supabase function
```

## 🛠️ Backend Infrastructure
- **Supabase Edge Function (`livekit-token`)**: Handles secure token generation for the frontend.
- **Python Agent SDK**: Provides the "brain" for real-time STT/TTS and LLM orchestration.
