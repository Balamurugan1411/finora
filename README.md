# 🚀 Finora – AI Money Mentor

> 🧠 An AI-powered financial decision engine that helps users **plan, predict, and optimize** their finances — not just track them.

---

## 📌 Problem Statement

95% of individuals lack a structured financial plan. Traditional financial advisors are expensive and inaccessible.

**Finora solves this by:**

* Providing **AI-driven financial guidance**
* Simplifying **complex financial decisions**
* Making planning as easy as chatting 💬

## 🏗️ Architecture (Agent-Based)

Finora uses an **AI agent pipeline**:

1. **Input Agent**

   * Collects user data (income, expenses, goals)

2. **Analysis Agent**

   * Processes financial data
   * Applies rules (tax slabs, SIP logic)

3. **Prediction Agent**

   * Forecasts future balance and risks

4. **Recommendation Agent**

   * Generates actionable insights

5. **Response Generator**

   * Converts output into human-friendly advice

```bash
# Install dependencies
cd finora
npm install

# Start backend
npm run server

# Start frontend
npm start
```
